# PortScan
Simple and fast port scanner built in Python

# Installing PortScan

Open terminal and paste the following commands :

```
git clone https://github.com/dimxz/PortScan
```
```
cd "PortScan"
```
```
pip install -r requirements.txt
```

# Usage

```
python PortScan.py
```
